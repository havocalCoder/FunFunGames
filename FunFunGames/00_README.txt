To Run in BlueJ: Open project in BlueJ. Run the main method from Main.java

To Run in Terminal: Compile all java files, run the main class

Content: The Main menu gives you access to four games, which you can play in order, and only play once. After playing all four games, you can play the fifth game, the Maze. Failing to complete the task or quitting halfway will cause you to lose all progress.
